What is Redstone?

Redstone is a group of fictional devices from the popular video game Minecraft. It serves as a loose analogue of electricity and can be used to create simple devices like light switches to more complicated things like combination locks or automated factories and machines.

Although each individual "redstone device" is simple, when combined they can become (to some extent) nearly as versatile as real-life electronic circuits. Players have even been able to create fully-functional — albeit simple — computers running inside of Minecraft!

This simulator lets you experiment with several redstone devices and create simple circuits in your web browser. It doesn't have every single tool and feature as the "real" Minecraft, but you can still use it to quickly try your ideas at any time.

Controls

  • You can move around by using the WASD and QE keys, and rotate the camera with the arrow keys.

  • The location that you're currently looking at is highlighted in yellow.

    Pressing Space will either remove the block at that location, or place a block when the location is empty.

  • You can change the block that will be placed by pressing the 1–4 keys. The yellow toolbar on the bottom shows which block that is currently selected.
